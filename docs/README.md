# Документация проекта

Актуальная документация собрана в этой папке.

## Разделы

- `START.md` — подробный старт: окружение, переменные, команды запуска.
- `PROJECT_OVERVIEW.md` — назначение проекта, архитектура и поток данных.
- `SWAGGER_API.md` — работа со Swagger/OpenAPI и проверка API вручную.

## Быстрый маршрут

1. Начни с `START.md`.
2. Затем прочитай `PROJECT_OVERVIEW.md`.
3. Для интеграции с API используй `SWAGGER_API.md`.
